# bank-app-search

Демонстрационный микросервис единого поиска в мобильном приложении банка.

## Возможности

- Полнотекстовый поиск по разделам (экранам) приложения
- Поиск по истории операций клиента через интеграцию с внешним сервисом `operation-history`
- История поисковых запросов клиента (последние 10 запросов)
- Экран подсказок с популярными разделами и недавними запросами

## Стек

- Java 21
- Spring Boot 3
- PostgreSQL
- Gradle

## API

| Метод | Путь                      | Описание                          |
|-------|---------------------------|-----------------------------------|
| POST  | `/api/v1/search`          | Выполнение поиска                 |
| POST  | `/api/v1/search/suggested` | Подсказки при открытии поиска    |
